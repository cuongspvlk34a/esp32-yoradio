✗ Partition Scheme "Default 4MB with spiffs" → SAI. Phải dùng Minimal SPIFFS (1.9MB APP with OTA/190KB SPIFFS). Tổng kích thước 11 file web chỉ ~29KB, hoàn toàn vừa trong 190KB SPIFFS.
✗ Cài thư viện IRremoteESP8266 → SAI. Thư viện này đã được tích hợp sẵn trong project tại src/IRremoteESP8266/. Cài thêm từ Library Manager sẽ gây xung đột và lỗi compile.
✗ #define XTASK_MEM_SIZE 4096 trong myoptions.h → KHÔNG CẦN. Project đã tự xử lý trong AsyncTCP.h với giá trị hợp lý. Không cần thêm dòng này vào myoptions.h.
✗ Dùng ESP32FS Plugin để upload SPIFFS → KHÔNG CẦN. Từ v0.9.000, firmware có WEB Board Uploader tích hợp sẵn. Upload qua trình duyệt sau khi nạp sketch.

Sơ đồ kết nối phần cứng Đọc kỹ trước khi cắm dây. Cắm sai có thể cháy module.
ℹ️

Cài đặt môi trường Arduino IDE Làm đúng thứ tự, không bỏ bước nào.
🚫
Arduino IDE 2.x.x KHÔNG được hỗ trợ. Phải dùng Arduino IDE 1.8.19.
ESP32 core 3.x chưa tương thích. Chỉ dùng core 2.x (ví dụ 2.0.17).
  • 1
    Tải Arduino IDE 1.8.19
    Vào arduino.cc/en/software → mục "Legacy IDE (1.8.X)"
  • 2
    Thêm ESP32 board URL
    File → Preferences → dán vào "Additional Boards Manager URLs": 
    https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  • 3
    Cài ESP32 core 2.x
    Tools → Board → Boards Manager → tìm "esp32 by Espressif" → chọn phiên bản 2.0.17 → Install
  • 4
    Cài đúng 4 thư viện — qua Library Manager
    Sketch → Include Library → Manage Libraries → tìm và cài từng cái: 
    Adafruit GFX Library
Adafruit ST7735 and ST7789 Libraries
OneButton
XPT2046_Touchscreen
    🚫
    KHÔNG cài các thư viện sau — đã built-in trong project:
    IRremoteESP8266 · ESP32-audioI2S · AsyncTCP · ESPAsyncWebServer · async-mqtt-client
    Cài thêm sẽ gây xung đột và lỗi compile ngay.
  • 5
    Thay file glcdfont.c — Bắt buộc
    Tìm file gốc trong thư mục Arduino libraries: 
    Windows: C:\Users\[tên]\Documents\Arduino\libraries\Adafruit_GFX_Library\glcdfont.c
macOS:   ~/Documents/Arduino/libraries/Adafruit_GFX_Library/glcdfont.c
    Thay bằng file từ project: 
    yoradio-main/yoRadio/fonts/glcdfont.c
    Sau đó khởi động lại Arduino IDE. Không thay file này → icon WiFi, signal, volume hiển thị ký tự lạ.

Chuẩn bị project & myoptions.h Cấu trúc thư mục phải đúng trước khi compile.
  • 1
    Giải nén yoradio-main.zip
    Cấu trúc thư mục cần có: 
    yoradio-main/
└── yoRadio/
    ├── yoRadio.ino       ← file chính, mở file này trong Arduino IDE
    ├── myoptions.h       ← ĐẶT VÀO ĐÂY, cùng thư mục với .ino
    └── data/
        ├── www/          ← 11 file web interface
        └── data/         ← playlist.csv, wifi.csv (để trống ban đầu)
  • 2
    Nội dung file myoptions.h — chính xác
    myoptions.h — đã xác minh qua source code
    #ifndef myoptions_h
#define myoptions_h

// Ngôn ngữ
#define L10N_LANGUAGE      EN

// Màn hình ST7789V 240x320 SPI
#define DSP_MODEL          DSP_ST7789
#define TFT_CS             5
#define TFT_DC             4
#define TFT_RST            15

// Cảm ứng XPT2046 (VSPI dùng chung)
#define TS_MODEL           TS_MODEL_XPT2046
#define TS_CS              16

// SD Card (VSPI dùng chung)
#define SDC_CS             17

// I2S DAC MAX98357
#define I2S_DOUT           27
#define I2S_BCLK           26
#define I2S_LRC            25
#define VS1053_CS          255  // tắt VS1053, dùng I2S DAC

// Encoder KY-040
#define ENC_BTNL           34   // CLK → vol-
#define ENC_BTNR           35   // DT  → vol+
#define ENC_BTNB           32   // SW  → nhấn nút
#define ENC_INTERNALPULLUP false

// IR Receiver IR1838
#define IR_PIN             36

#endif
    ⚠️
    Nếu encoder quay ngược chiều sau khi lắp: đổi chỗ 2 dây CLK↔DT trên board vật lý. Hoặc đổi ENC_BTNL 34 ↔ ENC_BTNR 35 trong file này rồi upload lại.

Cài đặt board & upload sketch Chú ý Partition Scheme — đây là điểm hay sai nhất.

Cài đặt trong Tools menu — phải đúng từng mục:

BoardESP32 Dev Module
CPU Frequency240MHz (WiFi/BT)
Flash Frequency80MHz
Flash ModeQIO
Flash Size4MB (32Mb)
Upload Speed921600
PSRAMDisabled
Core Debug LevelNone
🚫
Partition Scheme phải là: Minimal SPIFFS (1.9MB APP with OTA/190KB SPIFFS)
Tổng 11 file web chỉ ~29KB — vừa trong 190KB SPIFFS. Đây là scheme đúng đã kiểm tra thực tế.
  • 1
    Mở file yoRadio.ino trong Arduino IDE
    File → Open → chọn yoradio-main/yoRadio/yoRadio.ino
  • 2
    Upload sketch — nhấn Ctrl+U
    Lần đầu compile ~3–8 phút vì file Audio.cpp rất lớn (~221KB). Bình thường, không phải lỗi.
    ⚠️
    Nếu lỗi "Failed to connect": Giữ nút BOOT trên ESP32 → nhấn Upload → thả BOOT khi thấy dòng "Connecting..." xuất hiện.
5
Upload data qua WEB Board Uploader Không cần ESP32FS Plugin. Làm sau khi đã upload sketch.
✅
Từ v0.9.000, firmware tích hợp sẵn trang upload. Sau khi nạp sketch, ESP32 tự tạo WiFi AP và hiển thị trang upload ngay tại 192.168.4.1.
  • 1
    Kết nối vào WiFi AP của ESP32
    Tên WiFi APyoRadioAP
    Mật khẩu12345987
  • 2
    Mở trình duyệt → vào http://192.168.4.1
    Trang upload tự hiện. Nếu trình duyệt hỏi "không có internet có tiếp tục không" → chọn tiếp tục.
  • 3
    Chọn TẤT CẢ 11 file trong thư mục yoRadio/data/www/ → Upload
    Chọn tất cả cùng lúc (Ctrl+A), không upload từng file một. Danh sách 11 file: 
    dragpl.js.gz · ir.css.gz · ir.js.gz · irrecord.html.gz · logo.svg.gz
options.html.gz · player.html.gz · script.js.gz · style.css.gz
theme.css · updform.html.gz
    ⚠️
    Phải đủ cả 11 file. Thiếu updform.html.gz → chức năng update firmware không hoạt động.
  • 4
    ESP32 tự restart sau khi upload xong
    Bình thường. Sau restart, nếu chưa cấu hình WiFi → ESP32 tiếp tục ở AP mode để bạn nhập WiFi.
6
Cấu hình lần đầu (First Boot)
  • 1
    Nhập WiFi nhà
    Vẫn tại 192.168.4.1 → nhập SSID và password WiFi → Save → ESP32 tự kết nối và reboot.
  • 2
    Truy cập giao diện radio
    ESP32 hiển thị địa chỉ IP trên màn hình TFT. Mở trình duyệt → vào IP đó (ví dụ http://192.168.1.19).
  • 3
    Import playlist.csv
    Nhấn icon 🎵 (playlist) → tìm nút Import → chọn file playlist.csv → Upload.
    Hoặc vào Settings → WEB Board Uploader → upload playlist.csv cùng lúc với web files.
  • 4
    Settings → Timezone
    hours: 7 · minutes: 0 · NTP server: pool.ntp.org → nhấn APPLY.
  • 5
    Settings → Controls → IR Recorder (tùy chọn)
    Nhấn từng nút trên web → bấm nút tương ứng trên remote IR để gán chức năng.

✅
Khi mọi thứ hoạt động đúng: Màn hình ST7789V hiển thị tên đài, tên bài, đồng hồ, thời tiết. Xoay encoder để chỉnh âm lượng. Nhấn đôi để chuyển WEB/SD mode. Web interface tại IP của ESP32 để quản lý playlist và settings.
